## 📋 现在部署到 Vercel（5 分钟）

### 步骤 1：访问 Vercel（1 分钟）

1. 打开浏览器，访问：**https://vercel.com**
2. 点击右上角 **"Sign Up"** 或 **"Log In"**
3. 选择 **"Continue with GitHub"**
4. 授权 Vercel 访问你的 GitHub 账号

### 步骤 2：导入项目（1 分钟）

1. 登录后，点击 **"Add New..."** > **"Project"**
2. 在项目列表中找到 **`02-Ecommerce_React`**
3. 点击 **"Import"**

### 步骤 3：配置项目（1 分钟）

Vercel 会自动检测 Vite 项目，确认以下设置：

- ✅ **Framework Preset:** Vite（应该自动检测）
- ✅ **Root Directory:** `./`
- ✅ **Build Command:** `npm run build`
- ✅ **Output Directory:** `dist`
- ✅ **Install Command:** `npm install`

**直接点击 "Deploy" 旁边的 "Environment Variables"** 先配置环境变量

### 步骤 4：添加环境变量（重要！）

在部署前，点击 **"Environment Variables"**，添加：

```
名称: VITE_API_URL
值: https://strapi-store-server.onrender.com/api
环境: Production, Preview, Development (全选)

名称: VITE_TAX_RATE
值: 0.1
环境: Production, Preview, Development (全选)

名称: VITE_FREE_SHIPPING_THRESHOLD
值: 50000
环境: Production, Preview, Development (全选)

名称: VITE_SHIPPING_COST
值: 1000
环境: Production, Preview, Development (全选)
```

### 步骤 5：部署（2 分钟）

1. 点击 **"Deploy"**
2. 等待构建完成（约 2-3 分钟）
3. 看到 **"Congratulations!"** 表示部署成功
4. 点击部署链接（例如：`02-ecommerce-react.vercel.app`）
